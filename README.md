# Shopping App




This project is a mobile shopping app developed using Flutter. The purpose of this project is to showcase my learning journey with Flutter while building a functional and intuitive shopping app.

## Features




•	This shopping app includes the following features:


•	User authentication and account creation

•	Browse products by category and search for specific products

•	Add products to cart and view cart contents

•	Checkout and place an order

•	View order history and order details

### Technologies used




This project was developed using the following technologies:


•	Flutter SDK

•	Dart programming language


•	Firebase authentication, Firestore, and Cloud Functions
#Getting started




To run this app locally, follow these steps:


Clone this repository to your local machine:

[https://github.com/Barok-Getachew/Flutter_Learning_project.git]


**Navigate to the project directory:**


cd shopping-app


Install the required dependencies:


**flutter pub get**


Run the app:


**flutter run**
